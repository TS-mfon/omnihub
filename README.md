# OmniHub - Initia Cross-Minitia Asset Manager

A unified dashboard for managing assets across Initia's multi-VM Minitia ecosystem.

## Features

- **Portfolio Dashboard** - View total portfolio value across all Minitias
- **Cross-Minitia Bridge** - Transfer assets between Minitias seamlessly
- **Swap Interface** - Swap tokens across different Minitias
- **Yield Tracker** - Monitor yield positions across the ecosystem
- **Minitia Explorer** - Browse and discover all available Minitias
- **Activity Log** - Full transaction history across chains
- **Watchlist** - Monitor any wallet address
- **Alerts** - Set price and balance alerts

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- TanStack React Query
- Recharts

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
